# 🔐 Secret Language Coder & Decoder

### A fun python game which lets one text in a **Secret Language**. This program gives the functionality to encode a particular message and decode a message written in this language.

# 🚀 Features
- ### 🔁 Encode normal text into a secret language
- ### 🔓 Decode secret language back to original text
- ### 🎲 Randomized encoding for added complexity
- ### 🧠 Handles multiple words and sentences
- ### ⚠️ Strong input validation system
- ### 🔄 Continuous loop until user exits

# 🧩 How It Works

## 🔐 Encoding Rules:

### If a word has 3 or fewer characters:
- The word is reversed
### If a word has more than 3 characters:
- 3 random letters are added at the beginning and end
- The first 3 letters of the word are moved to the end

## 🔓 Decoding Rules:

### If a word has 3 or fewer characters:
- Reverse the word back
### If a word has more than 3 characters:
- Remove the added random letters
- Restore the original word structure

# ▶️ How to Run

- ### Make sure Python is installed
- ### Download or clone this repository
- ### Run the program: ```python main.py```

# 🎮 Usage

- ### Type:
  - ### ```E``` → Encode a message
  - ### ```D``` → Decode a message
  - ### ```Q``` → Quit the program
- ### Enter your phrase when prompted
- ### The program will display the encoded/decoded result
